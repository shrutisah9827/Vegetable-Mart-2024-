# Vegetable Mart

A lightweight static grocery storefront showcasing fresh vegetables and a browser-based shopping cart experience.

## Overview

This project includes a simple product catalog with category filters, a cart panel, and a feedback form. It is built with plain HTML, CSS, and JavaScript and runs entirely in the browser.

## Files

- `index.html` - Page structure and UI layout
- `styles.css` - Visual styling for pages, cards, tabs, and panels
- `app.js` - Product data, catalog rendering, cart logic, filtering, and feedback handling

## Features

- Product catalog with category tabs
- Live add-to-cart and cart summary
- Cart drawer with item count and total price
- Feedback form for saved reviews
- No backend required

## How to Run

1. Open `index.html` in a web browser.
2. Browse products and use the tabs to filter by category.
3. Add items to the cart and open the cart panel to review totals.
4. Submit feedback through the form.

## Notes

- The project is static and requires only a browser.
- Product data is stored in `app.js`.
- Feedback is handled locally in the page session.
